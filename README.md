# 💽 Wafflemate

TodoMate 어플을 참고하여 Wafflemate 어플을 제작했습니다.
<br/>
<br/>
기존 앱에 없는 **팔로우차단 기능**, **시간체크 기능**을 추가하여 SNS로써 활용도와 사용성을 높혔습니다.

## 👭"와플메이트" 는 어떤 앱인가요?

친구들과 함께하는 일정관리, 와플메이트!
<br/>
<br/>
일정관리와 더불어 팔로우 팔로잉이 가능한 SNS 기능을 제공합니다.
<br/>
일정/일기 공유를 통해 효과적인 일정관리를 경험하세요.
